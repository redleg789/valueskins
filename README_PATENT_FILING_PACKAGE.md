## 💡 PRO TIPS FOR SUCCESS

1. **File immediately** - Don't delay. Every day you wait, someone else might file similar invention.

2. **Keep filing receipt** - This number (application number) is critical. You'll need it for:
   - Checking status online
   - Filing full application
   - Any Patent Office correspondence
   - Priority claims for international filing

3. **Don't modify claims** - The claims are strategically broad. Modifications could weaken protection.

4. **Prepare for objections** - Most patent applications receive at least one office action (objection). This is NORMAL.
   - Save money in advance for attorney fees (~₹30,000) if examiner raises objections
   - Or learn to respond yourself (much cheaper)

5. **Think international** - If you plan to commercialize globally, file PCT application within 12 months:
   - Gives protection in 150+ countries
   - Cost: $2,000-5,000
   - Consider this before filing full Indian application

6. **Engage attorney later** - You don't need attorney for provisional filing, but consider one for full application/prosecution

7. **Keep copies organized** - Create folder:
