## SUBMISSION INSTRUCTIONS

**Online Submission (Recommended):**
1. Visit http://ipindiaonline.gov.in
2. Log in with credentials or create new account
3. Select "Patent" → "New Application" or "File Examination Request"
4. Upload all documents in PDF format
5. Complete Form 1A details in the system
6. Make online fee payment
7. Submit and save confirmation receipt

**Physical Submission (if required):**
1. Print Form 1A and all supporting documents
2. Prepare 2 copies (1 original + 1 duplicate)
3. Send via Speed Post to:
   - **The Patent Office (whichever is nearest)**
   - Chennai / Delhi / Kolkata / Mumbai Regional Offices
4. Keep Speed Post receipt and tracking details

**Note:** For latest filing guidelines and current fee structure, always check the official IP India website at http://www.ipindia.gov.in
